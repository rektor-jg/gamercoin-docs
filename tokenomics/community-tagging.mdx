CoinMarketCap and CoinGecko categorize tokens into tags (AI, DePIN, GameFi, Gaming, etc.). Those tags are how new users discover GHX when they browse a category — but tag updates only land when **community members** submit them. The team's own requests aren't enough.

This page gives you the exact form values to use. Each submission takes about **5 minutes**.

## What we're asking for

Three categories that match what GHX actually does today:

- **AI Applications** — GHX powers a live distributed-AI compute network.
- **DePIN** — the GamerHash GPU network is a textbook decentralized physical infrastructure project.
- **Gaming (GameFi)** — gaming and gamer-owned hardware are at the core of the ecosystem.

## CoinMarketCap

<Steps>
  <Step title="Open the request form">
    Go to the [CoinMarketCap Tag Request Page](https://support.coinmarketcap.com/hc/en-us/requests/new?ticket_form_id=360000553872).
  </Step>
  <Step title="Fill the form">
    Use these exact values:

    | Field | Value |
    | --- | --- |
    | Category | `7 - [Existing Cryptoasset] Update info (e.g. Rebrand, URL update, Tagging)` |
    | Subject | `GamerCoin - GHX - Update Tags` |
    | Project CMC URL | `https://coinmarketcap.com/currencies/gamercoin/` |
    | Tag Request Message | Paste the message provided in our community channels (link below). |
  </Step>
  <Step title="Submit">
    Click **Submit**. CMC reviews submissions before publishing — no further action needed from you.
  </Step>
</Steps>

## CoinGecko

<Steps>
  <Step title="Sign in and open the form">
    Sign into CoinGecko and open the [Coin/Token Information Update form](https://www.coingecko.com/request-form/coin-category-updates/new?locale=en).
  </Step>
  <Step title="Select the token">
    Search `GHX` and pick **GamerCoin** from the dropdown.
  </Step>
  <Step title="Set role and categories">
    | Field | Value |
    | --- | --- |
    | Submitter's role | `Community Member` |
    | Category 1 | `AI Applications` |
    | Category 2 | `DePIN` |
    | Category 3 | `Gaming (GameFi)` |

    For each category, paste the corresponding **Reason** text from our community channels.
  </Step>
  <Step title="Submit">
    Click **Submit**. CoinGecko also reviews each submission manually.
  </Step>
</Steps>

## Where to find the message templates

The exact text to paste for **Tag Request Message** (CMC) and **Reason 1/2/3** (CoinGecko) lives in pinned posts on:

- [Discord](https://discord.com/invite/T3ZpnwBvvC)
- [Telegram](https://t.me/GamerHashOfficial)

## Why this matters

Tag-driven discovery is one of the cheapest acquisition channels for any token. A user browsing the **DePIN** category on CoinGecko is already interested in distributed infrastructure — putting GHX in front of them is a much higher-quality intro than any ad. Every submitted request increases the odds these tags actually land.

Thanks for helping.
