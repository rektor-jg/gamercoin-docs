GamerCoin (GHX) is the native utility token of [GamerHash](https://gamerhash.com). It is used for payments, GPU compute rewards, staking, and liquidity programs.

## Quick facts

| | |
| --- | --- |
| **Symbol** | GHX |
| **Networks** | Ethereum (ERC-20), BNB Chain (BEP-20), Solana (SPL) |
| **TGE** | 31 December 2020 |
| **Circulating supply** | See [Distribution](/tokenomics/distribution) |
| **Whitepaper** | Registered with the Malta Financial Services Authority (MFSA) |
