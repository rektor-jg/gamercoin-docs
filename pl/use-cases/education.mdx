GamerHash uses GHX-denominated tasks to onboard users to crypto, exchanges, and Web3 mechanics. Tasks are completed inside the [Play&Earn](https://gamerhash.com/en/play-and-earn) platform.

## Task categories

| Category | Example tasks |
| --- | --- |
| Trading | Place a first GHX trade on a supported exchange |
| Staking | Stake GHX in a flexible pool |
| Wallets | Set up a wallet and receive GHX |
| Bridging | Move GHX between Ethereum, BSC, and Solana |
| Gaming | Download a featured title, create an account, play a tutorial |

Each task includes step-by-step instructions and a GHX or GHXP reward on completion.

## Why this exists

Distributed compute relies on contributors who can hold and use the network's token. The education program lowers the friction for users who are new to wallets, exchanges, and DeFi mechanics — letting them participate in the ecosystem without needing prior crypto experience.

## Where to find tasks

Active task lists are published in the [GamerHash AI App](https://gamerhash.com/pl/app-download) and on [gamerhash.com/en/play-and-earn](https://gamerhash.com/en/play-and-earn).
