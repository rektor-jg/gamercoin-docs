CoinMarketCap과 CoinGecko는 토큰을 태그(AI, DePIN, GameFi, Gaming 등)로 분류합니다. 이 태그는 신규 사용자가 카테고리를 탐색할 때 GHX를 발견하는 방법입니다 — 그러나 태그 업데이트는 **커뮤니티 회원**이 제출할 때만 적용됩니다. 팀 자체의 요청만으로는 부족합니다.

이 페이지는 양식에 사용할 정확한 값을 제공합니다. 각 제출은 약 **5분**이 걸립니다.

## 우리가 요청하는 것

GHX가 오늘 실제로 하는 일을 반영하는 세 가지 카테고리:

- **AI Applications** — GHX는 라이브 분산 AI 컴퓨팅 네트워크를 구동합니다.
- **DePIN** — GamerHash GPU 네트워크는 분산형 물리적 인프라 프로젝트의 교과서적인 예입니다.
- **Gaming (GameFi)** — 게이밍과 게이머 소유 하드웨어가 생태계의 핵심입니다.

## CoinMarketCap

<Steps>
  <Step title="요청 양식 열기">
    [CoinMarketCap 태그 요청 페이지](https://support.coinmarketcap.com/hc/en-us/requests/new?ticket_form_id=360000553872)로 이동하세요.
  </Step>
  <Step title="양식 작성">
    아래 정확한 값을 사용하세요:

    | 필드 | 값 |
    | --- | --- |
    | Category | `7 - [Existing Cryptoasset] Update info (e.g. Rebrand, URL update, Tagging)` |
    | Subject | `GamerCoin - GHX - Update Tags` |
    | Project CMC URL | `https://coinmarketcap.com/currencies/gamercoin/` |
    | Tag Request Message | 커뮤니티 채널에서 제공된 메시지를 붙여넣으세요(아래 링크). |
  </Step>
  <Step title="제출">
    **Submit**을 클릭하세요. CMC는 게시 전에 제출을 검토합니다 — 추가 작업이 필요 없습니다.
  </Step>
</Steps>

## CoinGecko

<Steps>
  <Step title="로그인 후 양식 열기">
    CoinGecko에 로그인하고 [코인/토큰 정보 업데이트 양식](https://www.coingecko.com/request-form/coin-category-updates/new?locale=en)을 여세요.
  </Step>
  <Step title="토큰 선택">
    `GHX`를 검색하고 드롭다운에서 **GamerCoin**을 선택하세요.
  </Step>
  <Step title="역할 및 카테고리 설정">
    | 필드 | 값 |
    | --- | --- |
    | Submitter's role | `Community Member` |
    | Category 1 | `AI Applications` |
    | Category 2 | `DePIN` |
    | Category 3 | `Gaming (GameFi)` |

    각 카테고리에 대해 커뮤니티 채널의 해당 **Reason** 텍스트를 붙여넣으세요.
  </Step>
  <Step title="제출">
    **Submit**을 클릭하세요. CoinGecko도 각 제출을 수동으로 검토합니다.
  </Step>
</Steps>

## 메시지 템플릿을 어디서 찾나

붙여넣을 정확한 텍스트(CMC의 **Tag Request Message**, CoinGecko의 **Reason 1/2/3**)는 다음 채널의 고정된 게시물에 있습니다:

- [Discord](https://discord.com/invite/T3ZpnwBvvC)
- [Telegram](https://t.me/GamerHashOfficial)

찾을 수 없다면 모더레이터에게 핑하세요 — 최신 사본을 DM으로 보내드립니다.

## 왜 중요한가

태그 기반 발견은 어떤 토큰에게도 가장 저렴한 획득 채널 중 하나입니다. CoinGecko에서 **DePIN** 카테고리를 탐색하는 사용자는 이미 분산 인프라에 관심이 있습니다 — 그들에게 GHX를 보여주는 것은 어떤 광고보다 훨씬 양질의 소개입니다. 제출되는 모든 요청은 이 태그가 실제로 적용될 확률을 높입니다.

도와주셔서 감사합니다.
