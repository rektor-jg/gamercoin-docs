| Year | Milestone |
| --- | --- |
| Nov 2017 | GamerHash Miner App launch |
| Mar 2018 | 150,000 registered users |
| Aug 2018 | ~300,000 registered users |
| 2019–2020 | GHX token development and pre-launch |
| 31 Dec 2020 | GHX Token Generation Event |
| Apr 2021 | First exchange listing (Bithumb) |
| 2021–2022 | Seven token burn rounds; 8.1% of supply removed |
| 2023–2024 | Pivot to GamerHash AI App; GPU sharing for AI workloads |
| 2025 | deAPI public launch; Solana bridge; expanded AI model coverage |

GamerHash started as a mining client that paid users for idle GPU time. The strategic direction shifted as open-source AI models created demand for distributed GPU compute outside of crypto mining. The platform now monetizes GPU contribution across both AI inference workloads and traditional Proof-of-Work mining.

Today the platform has close to 800,000 registered users.
