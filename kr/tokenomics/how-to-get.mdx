## Inside the GamerHash platform

| Method | How it works |
| --- | --- |
| **GPU sharing** | Run the [GamerHash AI App](https://gamerhash.com/pl/app-download) and earn GHX for contributing GPU compute |
| **GHXP conversion** | Convert accumulated GamerHash Experience Points (GHXP) into GHX |
| **Community campaigns** | Airdrops, contests, and partner promotions |
| **Staking & LP rewards** | Earn GHX by participating in [staking](/tokenomics/staking) or [liquidity mining](/tokenomics/liquidity-mining) |

GHX earned in-platform credits to the user wallet. From there it can be withdrawn to an external wallet or spent in the [GamerHash Store](https://gamerhash.com/en/shop).
